## データフローの例

```
1. ユーザーが「東京 vs ロンドン比較」をクリック
                    │
                    ▼
2. フロントエンドがAPIを呼び出し: GET /cost_of_living?city1=Tokyo&city2=London
                    │
                    ▼
3. FastAPIがdata/processed/cost_of_living.jsonを読み込み
                    │
                    ▼
4. 比較データをフロントエンドに返す
                    │
                    ▼
5. フロントエンドがチャートと表を表示
```

## 技術スタック

### データパイプライン（Python）
- **Luigi**: データ収集タスクを管理
- **BeautifulSoup**: ウェブサイトをスクレイピング
- **Pandas**: データ処理とクリーニング
- **Schedule**: パイプラインの定期実行

### バックエンドAPI（Python）
- **FastAPI**: モダンウェブフレームワーク
- **Pydantic**: データバリデーション
- **Uvicorn**: ASGIサーバー
- **Python 3.11+**: 最新機能

### フロントエンド（JavaScript/TypeScript）
- **React**: UIフレームワーク
- **TypeScript**: 型安全性
- **Vite**: ビルドツール
- **Chart.js**: データ可視化
- **Tailwind CSS**: スタイリング

### インフラ
- **バックエンドデプロイ**: Google Cloud Run
- **フロントエンドデプロイ**: Vercel
- **データストレージ**: ローカルJSON/Parquetファイル
- **バージョン管理**: Git + GitHub
