## Data Flow Example

```
1. User clicks "Compare Tokyo vs London"
                    │
                    ▼
2. Frontend calls API: GET /cost_of_living?city1=Tokyo&city2=London
                    │
                    ▼
3. FastAPI reads data/processed/cost_of_living.json
                    │
                    ▼
4. Returns comparison data to Frontend
                    │
                    ▼
5. Frontend displays charts and tables
```

## Technology Stack

### Data Pipeline (Python)
- **Luigi**: Orchestrates data collection tasks
- **BeautifulSoup**: Scrapes websites
- **Pandas**: Processes and cleans data
- **Schedule**: Runs pipeline periodically

### Backend API (Python)
- **FastAPI**: Modern web framework
- **Pydantic**: Data validation
- **Uvicorn**: ASGI server
- **Python 3.11+**: Latest features

### Frontend (JavaScript/TypeScript)
- **React**: UI framework
- **TypeScript**: Type safety
- **Vite**: Build tool
- **Chart.js**: Data visualization
- **Tailwind CSS**: Styling

### Infrastructure
- **Backend Deploy**: Google Cloud Run
- **Frontend Deploy**: Vercel
- **Data Storage**: Local JSON/Parquet files
- **Version Control**: Git + GitHub
